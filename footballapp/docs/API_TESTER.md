FootballApp Developer Utilities

ApiTester

Purpose
-------

Used to inspect and validate API-Football endpoints before
creating model classes and import services.

Workflow
--------

1. Explore endpoint in API Dashboard.

2. tester.print()

3. tester.save()

4. tester.validate()

5. Build model classes.

6. Build import service.

7. Build repository.

8. Build REST endpoint.

9. Build Android screen.
